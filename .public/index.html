Change Detection Monitoring Service
WORLD DATA CENTER FOR GEOINFORMATICS AND SUSTAINABLE DEVELOPMENT
Entry points
Fetch data as geojson
  • Fetch all monitoring data:
    https://wdc-cdms.herokuapp.com/api/json
  • Fetch filtered monitoring data by measurement date:
    https://wdc-cdms.herokuapp.com/api/json?measurement_date="2021.01.06"
  • Fetch filtered monitoring data by measurement date range:
    https://wdc-cdms.herokuapp.com/api/json?measurement_date=["2021.01.06","2021.05.19"]
  • Fetch filtered monitoring data by measurement type:
    https://wdc-cdms.herokuapp.com/api/json?measurement_type="nominal"
  • Fetch filtered monitoring data by object name:
    https://wdc-cdms.herokuapp.com/api/json?Name="Симферопільське"
  • Fetch monitoring data without geometry:
    https://wdc-cdms.herokuapp.com/api/json?measurement_type="nominal"&Name="Симферопільське"&excludeGeometry=1
  • Fetch selected properties:
    https://wdc-cdms.herokuapp.com/api/json?measurement_type="nominal"&Name="Симферопільське"&excludeGeometry=1&properties=["OBJECTID", "Name", "Size"]
